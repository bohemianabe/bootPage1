Travel Site

Each page has slide animations as the user scrolls down the page, and inbetween page transitions is a swipe of colors to make a smooth transition, and enhance the user experience.

Todo List

My "Todo List" app is a dynamic webpage that takes a user's input (will prompt a error for an empty string entry), and list them in a row beneath it. Each row has the ability to mark an item complete, or to be deleted. Beside the entry list is a drop down button to filter out options of: 'All', 'Complete', or 'Incomplete'. The list is stored in local-storage so it won't be lost when the page is refreshed, and can be removed when deleted. What are you waiting for? Make your todo list, and become the most productive human being on earth!
To see it's full functionality please visit the link here!

Beat Maker

"Beat maker" is a dynamic app for mixing beats. Upon landing on the page the user will have three tracks to chose from: kick, snare, and hihat. Each have seven options of unique sounds to pick from. The user can pick their rhythm along the eight pads beside each option. Below the third track is a dynamic play button, and a tempo-slider that has a minimum of 20 bpm to a maximum of 300 bpm. Each track can be muted by using the mute button beside the select button. Give your hand at mixing some beats, and check out it's functionality by visiting the link here.

Jamming

Jamming is a fully functional App linked with a spotify API key to create and update playlists on your own spotify account! It built using the React framwork, and should add a little flavor to your playlist! It has two columns one to search for potential new music, and the second to adjust your playlist.
